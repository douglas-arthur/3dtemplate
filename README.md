This tool allows you to view uploaded pngs in a 3d exploded view.  
The interface allows you to expand the layers of seperation, center, and rotate like the Ironman schematics.  
The UI can be hidden to take screenshots.

I built this as a supplemental tool for my CIRCLR app where people can make concentric circle patterns and then directly upload them at my website https://www.circlr.art/3dtemplate
